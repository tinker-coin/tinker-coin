## Running unit-tests
### Running Unit tests only.
```
tinker-node> pytest
```
### Running Unit tests with coverage.
```
tinker-node> coverage run -m pytest
```
### See quick coverage report
```
tinker-node> coverage report
```
### Generate coverage html report
```
tinker-node> coverage html
```
